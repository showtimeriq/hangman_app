# Description 

Your classic word guessing game built with Ruby on Rails, data stored with Postgresql db and hosted on Heroku! You can create a sharable link to play with your friends. Single-player support coming soon using word bank generated with [Spicy Proton](https://github.com/schmich/spicy-proton) gem. 

# Local Development 
* git clone https://github.com/showtimeriq/hangman_app.git
* bundle install 
* bundle exec rake db:create db:migrate
* rails s
* visit localhost:3000
